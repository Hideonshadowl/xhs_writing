# 帖子共创模板

## 用户输入

不要求每项都提供，有什么就给什么。

- 本次想推广的文稿类型：
- 目标读者（优先从主要受众里选）：基层公务员 / 选调生 / 社区网格员 / 国企或银行员工 / 年轻老师 / 讲解员 / 宣讲员 / 部队文职 …
- 想讲的案例或问题：
- 已有素材：
- 希望呈现的专业能力：
- 希望避免的内容：
- 是否有发布日期或节点：
- 其他要求：

## 默认输出

### 1. 标题

提供最终标题，并可附 2—3 个备选。优先「身份或体裁 + 具体卡点」，例如：

- `社区网格员的工作总结应该怎么写？`
- `调研报告怎么写出亮点？`
- `微党课怎么让台下从昏昏入睡到聚精会神？`
- `小科员也能写出大文章`

### 2. 正文

输出一版可直接发布的正文。

### 3. 话题标签

提供最多 5 个相关标签（双平台上限），不堆砌无关热门词。

### 4. 小红书封面图

提供一张封面图（常见 3:4），并给出封面文案（8—14 字）。

### 5. 抖音 slides

提供一套 **9:16（1080×1920）** 纯图片图文 slides，铺满竖屏、无上下黑边。归档路径示例：`assets/slides-主题/`。

### 6. 归档

文件名格式：

`posts/YYYY-MM-DD-帖子主题.md`

归档内容包括：

- 标题最终版；
- 小红书封面图路径与封面文案；
- 抖音 slides 路径与比例（9:16）；
- 正文；
- 标签；
- 发布状态；
- 数据复盘。

同一选题文案可双平台复用；抖音必须另交 9:16 slides。

## 快速指令示例

> 用刚才那篇“匠心工具箱”党课作为脱敏案例，写一篇长文。重点体现我能把三份散乱材料整合成10分钟完整脚本，不公开客户和单位信息，语气专业但不要太像广告。配一张封面。
