# 账号定位

## 一句话定位

帮助没时间、没思路或不擅长表达的职场人，高效完成重要文稿。

## 双平台分工

小红书与抖音统一发长文图文：一篇正文 + 一张封面图。不做视频。

### 小红书

- 用完整图文承接搜索，沉淀写作方法和脱敏案例；
- 让读者通过主页与历史内容确认专业度；
- 适合 500—900 字的案例拆解和方法说明。

### 抖音

- 用同一篇长文获取场景化曝光与搜索流量；
- 封面突出冲突或结果，正文保留完整逻辑；
- 用评论区和主页内容承接需求，不在正文里反复硬推销。

同一选题双平台共用一套内容，按平台字数与标签习惯微调即可。

## 主要引流受众

这些人群是账号的核心引流对象，也是最常提写作、写稿、写材料需求的人：

| 人群 | 常见文稿需求 |
| --- | --- |
| 基层公务员 / 小科员 | 讲话稿、汇报、调研报告、工作总结、经验材料 |
| 基层选调生 | 调研报告、发言稿、工作总结、汇报材料 |
| 社区员工（含网格员） | 工作总结、述职、汇报、发言稿、活动材料 |
| 国企员工 / 银行职员 | 演讲稿、宣讲稿、汇报、总结、先进事迹材料 |
| 年轻老师 | 演讲稿、发言稿、宣讲稿、主题教育相关讲稿 |
| 展示馆讲解员 | 讲解词、现场教学讲稿、宣讲稿 |
| 宣讲员 | 宣讲稿、微党课、情景党课、主题演讲 |
| 部队文职（辅导员、讲解员） | 辅导讲稿、讲解词、宣讲稿、情景课脚本 |

标题和封面要让上述身份**一眼对上号**，优先「身份 / 体裁 + 具体卡点」，少写空泛「职场人」「打工人」。

## 重点承接需求

- 讲话稿、发言稿、汇报材料、调研报告；
- 党课、微党课、情景党课、演讲稿、宣讲稿、讲解词；
- 工作总结、述职报告、经验材料；
- 活动方案、主持词、新闻稿、公众号稿；
- 文稿修改、压缩、润色和结构重写。

## 内容支柱

### 1. 痛点共鸣

呈现真实岗位场景，例如“网格员年终总结又写成流水账”“小科员第一次写调研报告”，让目标客户快速确认账号与自己有关。

### 2. 写作方法

分享结构、开头、标题、金句、修改方法和常见问题，建立专业可信度。

### 3. 脱敏案例

展示接到需求后如何分析、搭框架、处理冲突和形成成稿。重点讲思路与变化，不公开客户信息和完整敏感材料。

### 4. 服务展示

说明可以解决哪些文稿问题、合作需要哪些材料、交付流程如何进行，降低咨询门槛。

### 5. 职场观察

围绕写材料、临时任务、跨部门沟通和领导修改意见等话题表达理解，增强账号人格感。

## 账号语气

- 专业，但不端着；
- 懂工作场景，但不冒充体制内身份；
- 有方法，也有人情味；
- 不贬低客户写作能力，不制造羞耻感；
- 少用夸张营销词，多用具体问题和实际解决方案。

## 转化路径

长文解决一个小问题  
→ 读者通过主页和历史内容确认专业度  
→ 评论或私信描述需求  
→ 明确用途、受众、篇幅、截止时间和现有素材  
→ 评估后承接并交付

## 内容边界

- 所有案例默认去除单位、姓名、地点、项目及内部数据。
- 不接触国家秘密、工作秘密或未经授权的内部敏感材料。
- 不代写考试、考核中明确要求本人独立完成的内容。
- 不发布带有政治常识错误、政策表述风险或虚假事实的内容。
- 涉及政策和重要数据时，以权威公开来源为准。
